سرفصل‌های دوره پایتون مقدماتی

معرفی پایتون

پایتون:

پایتون یک زبان برنامه‌نویسی محبوب و همکاره است که گیدو ون روسوم در ۱۹۹۱ آن را توسعه داد.

چند ویژگی اصلی پایتون:

  • برای برنامه‌های وب، اتوماسیون، کار با پایگاه داده و داده‌های بزرگ کاربرد دارد.
  • سینتکس ساده و خوانا دارد.
  • از برنامه‌نویسی رویه‌ای، شی‌گرا و تابعی پشتیبانی می‌کند.

شروع پایتون

نصب پایتون:

برای بررسی اینکه پایتون روی سیستم شما نصب است، از دستورات زیر در خط فرمان استفاده کنید:

import sys
print(sys.version)

اگر پایتون نصب نیست، می‌توانید آخرین نسخه را از سایت رسمی python.org دانلود کنید.

بررسی نسخه پایتون:

برای بررسی نسخه پایتون در محیط اجرا:


print("Hello, World!")

نوشتن فایل پایتون:

پایتون یک زبان تفسیری است، به این معنی که شما فایل‌ پایتون را با پسوند`.py` را نوشته و سپس اجرا می‌کنید.

مثال:

print("Hello, World!")

برای اجرا:

python helloworld.py

کامنت ها

کامنت ها:

  • از کامنت می توان برای توضیح کد پایتون استفاده کرد.
  • برای خوانایی بیشتر کد می توان از کامنت استفاده کرد.
  • از کامنت می توان برای جلوگیری از اجرا در هنگام تست کد استفاده کرد.

ایجاد یک کامنت:

کامنت با # شروع می شود و پایتون آنها را نادیده می گیرد:

# This is a comment
print("Hello, World!")

متغیرها

متغیرها (variables) برای ذخیره داده‌ها استفاده می‌شوند.

x = 10
y = "Python"

تعریف متغیرها در پایتون

متغیرها در پایتون به‌طور خودکار هنگام مقداردهی اولیه ایجاد می‌شوند.

x = 5
y = "Hello"
print(x)
print(y)

تغییر نوع داده‌ی متغیر

می‌توان نوع داده را با casting مشخص کرد:

x = str(3)  # تبدیل به رشته: '3'
y = int(3)  # تبدیل به عدد صحیح: 3
z = float(3)  # تبدیل به عدد اعشاری: 3.0

نام‌گذاری متغیرها

مجاز:

  • باید با حرف یا _ شروع شوند.
  • می‌توانند شامل اعداد، حروف و _ باشند.

    • مثال

    myvar = "John"
my_var = "John"
_my_var = "John"
myVar = "John"
MYVAR = "John"
myvar2 = "John"

غیرمجاز:

  • شروع با عدد (مثل 1variable).
  • استفاده از کاراکترهای ویژه مثل @, $, %.

    • مثال

    2myvar = "John"
my-var = "John"
my var = "John"

مقداردهی چندگانه

پایتون به شما این امکان را می دهد که در یک خط به چندین متغیر مقادیر اختصاص دهید:

x, y, z = "Apple", "Banana", "Cherry"
print(x)
print(y)
print(z)

مقادیر مشترک

و می توانید یک مقدار را به چندین متغیر در یک خط اختصاص دهید:

x = y = z = "Orange"
print(x, y, z)  # Orange Orange Orange

حساسیت به حروف بزرگ و کوچک

متغیرها در پایتون به حروف بزرگ و کوچک حساس هستند.

a = 4
A = "Sally"
a و A دو متغیر متفاوت هستند

دریافت نوع متغیر

می‌توانید نوع متغیر را با تابع type() مشخص کنید.

x = 5
y = "John"
print(type(x))
print(type(y))

خروجی متغیرها

تابع print() برای نمایش مقادیر متغیرها استفاده می‌شود.

x = "Python is awesome"
print(x)

می‌توان چند متغیر را با جداکننده کما چاپ کرد:

x = "Python"
y = "is"
z = "awesome"
print(x, y, z)

یا با اپراتور + متغیرها را به هم متصل کرد:

x = "Python "
y = "is "
z = "awesome"
print(x + y + z)

برای عددها، اپراتور + به عنوان عملگر ریاضی عمل می‌کند:

x = 5
y = 10
print(x + y)

انواع داده ها

انواع داده های داخلی

در برنامه نویسی، نوع داده یک مفهوم مهم است.

متغیرها می توانند داده های مختلف را ذخیره کنند و انواع مختلف می توانند کارهای متفاوتی انجام دهند.

پایتون انواع داده های زیر را به طور پیش فرض در این دسته بندی ها داخلی دارد:

نوع متن: str

نوع عددی: int, float, complex

انواع توالی: list, tuple, range

نوع نقشه برداری: dict

انواع مجموعه: set, frozenset

نوع بولی: bool

نوع باینری: bytes, bytearray, memoryview

نوع هیچ: none

 
a = 5
b = 3
print(a + b)  # خروجی: 8

دریافت نوع داده:

با استفاده از تابع type() می توانید نوع داده هر شی را بدست آورید:

x = 5
print(type(x))
# خروجی int

اعداد

اعداد پایتون

سه نوع عددی در پایتون وجود دارد:

int, float, complex

متغیرهای انواع عددی زمانی ایجاد می شوند که یک مقدار به آنها اختصاص دهید:

x = 12     # int
y = 220.8  # float
z = 1j     # complex

تبدیل نوع داده ها

تبدل نوع:

می توانید با متدهای int()، float() و complex() از یک نوع به نوع دیگر تبدیل کنید:

int, float, complex

 
x = 1    # int
y = 2.8  # float
z = 1j   # complex

#convert from int to float:
a = float(x)

#convert from float to int:
b = int(y)

#convert from int to complex:
c = complex(x)

print(a)
print(b)
print(c)

print(type(a))
print(type(b))
print(type(c))

رشته ها

رشته‌ها - strings در پایتون توسط یک علامت نقل قول یا دو علامت نقل قول احاطه شده‌اند.

"Hello" همان 'hello' است

print("Hello")
print('Hello')

نقل قول در داخل نقل قول:

شما می توانید از نقل قول ها در داخل یک رشته استفاده کنید، به شرطی که با نقل قول های اطراف رشته مطابقت نداشته باشند:

print("It's alright")
print("He is called 'Johnny'")
print('He is called "Johnny"')

رشته های چند خطی:

با استفاده از سه نقل قول می توانید یک رشته چند خطی را به یک متغیر اختصاص دهید:

a = """Lorem ipsum dolor sit amet,
consectetur adipiscing elit,
sed do eiusmod tempor incididunt
ut labore et dolore magna aliqua."""
print(a)

طول رشته:

برای بدست آوردن طول یک رشته، از تابع len() استفاده کنید.

a = "Hello, World!"
print(len(a))

برش دادن:

با استفاده از نحو slice می توانید طیف وسیعی از کاراکترها را برگردانید.

b = "Hello, World!"
print(b[2:5])

الحاق رشته:

برای به هم پیوستن یا ترکیب دو رشته می توانید از عملگر + استفاده کنید.

a = "Hello"
b = "World"
c = a + b
print(c)

قالب رشته F-string

F-String در پایتون 3.6 معرفی شد و اکنون روش ترجیحی برای قالب بندی رشته ها است.

برای تعیین یک رشته به‌عنوان یک رشته f، به سادگی یک f را در مقابل رشته واقعی قرار دهید و براکت‌های {} را به‌عنوان مکان‌نما برای متغیرها و سایر عملیات اضافه کنید.

age = 36
txt = f"My name is John, I am {age}"
print(txt)

متدها - Methods

رشته‌ها در پایتون دارای متدهای داخلی زیادی هستند که می‌توانند برای تغییر یا بررسی رشته‌ها استفاده شوند.

تبدیل به حروف بزرگ و کوچک

برای تبدیل تمام حروف یک رشته به حروف بزرگ یا کوچک، از متدهای زیر استفاده می‌شود:

txt = "Hello, World!"
print(txt.upper())  # تبدیل به حروف بزرگ
print(txt.lower())  # تبدیل به حروف کوچک

حذف فاصله‌ها

برای حذف فاصله‌های اضافی از ابتدا و انتهای یک رشته، از متد strip() استفاده می‌شود:

txt = "   Hello, World!   "
print(txt.strip())  # حذف فاصله‌های اضافی

جایگزینی کاراکترها

متد replace() برای جایگزینی بخش خاصی از یک رشته استفاده می‌شود:

txt = "Hello, World!"
print(txt.replace("H", "J"))  # جایگزینی H با J

تقسیم رشته‌ها

متد split() برای تقسیم یک رشته بر اساس یک جداکننده مشخص استفاده می‌شود:

txt = "Hello, World!"
print(txt.split(","))  # تقسیم رشته بر اساس کاما

بررسی وجود زیررشته

برای بررسی اینکه آیا یک زیررشته خاص درون یک رشته قرار دارد یا خیر، از عملگر in استفاده می‌شود:

txt = "Hello, World!"
print("World" in txt)  # بررسی وجود "World" در رشته

بررسی عدم وجود زیررشته

برای بررسی اینکه آیا یک زیررشته درون رشته موجود نیست، از عملگر not in استفاده می‌شود:

txt = "Hello, World"
print("Python" not in txt)
# بررسی عدم وجود "Python" در رشته

حلقه‌ها و شرط‌ها

حلقه‌ها برای تکرار و شرط‌ها برای تصمیم‌گیری استفاده می‌شوند.

for i in range(5):
                          print(i)

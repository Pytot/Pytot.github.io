سرفصل‌های دوره پایتون مقدماتی

معرفی پایتون

پایتون یک زبان برنامه‌نویسی سطح بالا و همه‌منظوره است که برای توسعه وب، تحلیل داده‌ها، هوش مصنوعی و موارد دیگر استفاده می‌شود.

print("Hello, World!")

متغیرها و انواع داده‌ها

در پایتون، متغیرها برای ذخیره داده‌ها استفاده می‌شوند. انواع داده‌ها شامل اعداد، رشته‌ها، لیست‌ها و غیره هستند.

x = 10
y = "Python"

عملگرها

عملگرها برای انجام عملیات‌های ریاضی، مقایسه‌ای و منطقی استفاده می‌شوند.

a = 5
b = 3
print(a + b)  # خروجی: 8

لیست‌ها

لیست‌ها برای ذخیره مجموعه‌ای از آیتم‌ها استفاده می‌شوند و قابل تغییر هستند.

fruits = ["apple", "banana", "cherry"]
print(fruits[0])  # خروجی: apple

حلقه‌ها و شرط‌ها

حلقه‌ها برای تکرار و شرط‌ها برای تصمیم‌گیری استفاده می‌شوند.

for i in range(5):
    print(i)
